# My Dotfiles Script



## Ansible

Instalar o Ansible 
```shell
sudo dnf install ansible -y  # Fedora
```

Instalar os collection
```shell
ansible-galaxy install -r playbook/requirements.yml
```

instalar todos os apps/ferramentas
```shell
sudo ansible-playbook playbook/tasks.yml
```
